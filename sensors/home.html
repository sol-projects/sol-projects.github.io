SOL Sensors

SOL Sensors is a collection of programs and libraries for Windows and Linux meant for fetching information from hardware sensors on Windows and Linux.

Functionality

Linux

- [✓] CPU usage
- [✓] CPU temperature
- [✓] CPU Threads usage
- [✓] RAM usage
- [x] RAM temperature
- [✓] Nvidia GPU usage
- [✓] Nvidia GPU temperature
- [✓] Nvidia GPU VRAM usage
- [✓] Nvidia GPU VRAM temperature
- [x] AMD GPU usage
- [x] AMD GPU temperature
- [x] AMD GPU VRAM usage
- [x] AMD GPU VRAM temperature

Windows

- [x] CPU usage
- [x] CPU temperature
- [✓] CPU Threads usage
- [✓] RAM usage
- [x] RAM temperature
- [✓] Nvidia GPU usage
- [✓] Nvidia GPU temperature
- [✓] Nvidia GPU VRAM usage
- [✓] Nvidia GPU VRAM temperature
- [x] AMD GPU usage
- [x] AMD GPU temperature
- [x] AMD GPU VRAM usage
- [x] AMD GPU VRAM temperature

QT Program

The GUI version of the program displays realtime graphs for hardware components. To compile and run the QT GUI version of the program, you must have Qt5 or Qt6 installed. You can find more information on the [website](https://sol-projects.github.io/sensors/tutorial/gui.html)
