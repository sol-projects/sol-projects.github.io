Building manually

Requirements:
- Linux or MacOS
- Qt5 or Qt6 - CMake version 3.18 - 3.20
- CXX 20 compliant Compiler
- Optional Requirements:
- an internet connection
- Git

Example: 
cmake -S . -B Debug
cmake --build Debug
cd Debug
./ProjectName

Examples

The program displays graphs in realtime. You can toggle them on/off on the left panel.
